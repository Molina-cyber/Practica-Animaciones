Transiciones con css

Permiten cambiar suavemente una propiedad CSS de un estado a otro durante un periodo de tiempo

Propiedades de las transiciones

  • transition-property: Define que propiedad cambiar
  • trasition-duration: Define la duracion del cambio. (s o ms)
  • transition-timing-function: Controla la aceleracion o desaceleracion.
    • linear -> Velociad constante.
    • ease (por defecto) -> comienza lenta, acelera y luego desacelera
    • ease-in -> comienza lenta y acelera
    • ease-out -> comienza rapido y desacelera
    • ease-in-out -> comienza rapido y termina lenta
  • transition-delay: Retrasa el inicio de la transicion
