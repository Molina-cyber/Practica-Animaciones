Transiciones con css

Permiten cambiar suavemente una propiedad CSS de un estado a otro durante un periodo de tiempo

Propiedades de las transiciones

  • transition-property: Define que propiedad cambiar
  • trasition-duration: Define la duracion del cambio. (s o ms)
  • transition-timing-function: Controla la aceleracion o desaceleracion.
    • linear -> Velociad constante.
    • ease (por defecto) -> comienza lenta, acelera y luego desacelera
    • ease-in -> comienza lenta y acelera
    • ease-out -> comienza rapido y desacelera
    • ease-in-out -> comienza rapido y termina lenta
  • transition-delay: Retrasa el inicio de la transicion

Transicion de color

La transicion de color de fondo en CSS permite cambiar suavemente el color de fondo de un elemento cuando se activa un cambio de estado, como un :hover. Esto crea un efecto visual agradable y fluido que mejora la experiencia del usuario al interactuar con la pagina. Por ejemplo, al pasar el cursor sobre un boton, su color de fondo puede cambiar de azul a rojo, atrayendo la atencion del usuario sin ser abrupto

Animaciones

Permiten que los elementos cambien de estilo en varios estados definidos mediante @keyframes

Propiedades basicas de las animaciones

  • animation-name: Especifica el nombre de la animacion(@keyframes).
  • animation-duration: Duracion de la animacion (s o ms)
  • animation-timing-function: Controla la aceleracion o desaceleracion
    • linear -> Velocidad constante
    • ease (por defecto) -> Comienza lenta, acelera y luego desacelera
    • ease-in -> Comienza lenta y acelera
    • ease-out -> Comienza rapido y desacelera
    • ease-in-out -> Comienza rapido y termina lenta
  • animation-delay: Retrasa el inicio de la animacion
  • animation-iteration-count: Numero de veces que se repite la animacion (infinite para infinito)
  • animation-direction: Direccion de la animacion (normal, reverse, alternate, alternate-reverse)
  • animation-fill-mode: Controla el estado del elemento antes y despues de la animacion

Animacion de movimiento

La animacion de movimiento en CSS utiliza la propiedad transform junto @keyframes para mover elementos a lo largo del eje X o Y, creando un efecto de desplazamiento. Este tipo de animaciones es ideal para captar la atencion del usuario y hacer que la interfaz sea mas dinamica. Por ejemplo, un cuadro puede moverse horizontalmente de un lado a otro de la patalla, lo que puede usarse para destacar informacion o guiar la mirada del usuario hacia un elemento importante.
